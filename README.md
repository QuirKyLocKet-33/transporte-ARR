# Instrucciones para deploy en Vercel

1. Todo el contenido público está en la carpeta `/public`.
2. El archivo `vercel.json` configura el despliegue como sitio estático.
3. Los archivos originales están fuera de `/public` para mantener el orden.
4. Sube este proyecto a GitHub y conéctalo a Vercel.
5. Vercel servirá los archivos desde `/public`.
